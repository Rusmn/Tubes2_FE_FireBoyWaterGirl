# TUBES2_FIREBOYWATERGIRL

## Menjalankan

1. Clone repo ini:
   ```bash
   git clone https://github.com/Rusmn/Tubes2_FE_FireBoyWaterGirl.git
   ```

2. Install semua dependency:
   ```bash
   npm install
   ```

3. Jalankan server:
   ```bash
   npm run dev
   ```

Buka di browser: [http://localhost:5173](http://localhost:5173)
